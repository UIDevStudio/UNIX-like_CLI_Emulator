Создание нового релиза:

1. Создаешь новый тег. Вкладка "Select tag". Наименование тега: terminal_v1.3

2. Выбираешь название. А именно: GUI Terminal. DA: Python Edition, [Version 1.3]

3. Выбираешь на вкладке "Previous tag" предыдущий тег: terminal_v1.2

4. Заполняешь поле для текста. Это описание релиза. Текст будет прикреплен отдельно.

5. Выбираешь исполняемый файл для релиза. Прикрепляешь его в поле "Attach binaries by dropping them here or selecting them".

6. В самом конце внизу находишь зеленую кнопку "Publish release" и нажимаешь на нее.

------------------------------

Редактирование README.md:

1. Нажимаешь значок карандаша в правом верхнем углу в README.md (режим редактирования).

2. Копируешь строку с последним релизом 1.2 и вставляешь на ней.

3. В новой строке меняешь обновление на 1.3 и меняешь в ссылке цифры (с v1.2 на v1.3).

4. В столбце "Версия" **v1.2** меняешь на **v1.3**.

5. В строке с обновлением 1.2 меняешь **v1.2** на v1.2 и вместо словосочетания "Последнее обновление" пишешь "Устаревшая версия".

6. Коммитишь изменения. Нужно написать в поле "Commit message" следующее: Обновил README. Автор: Xelmor.

------------------------------

Финальный штрих: отписываешься мне, когда закончишь, чтобы я проверил изменения.
